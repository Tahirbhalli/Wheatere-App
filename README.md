# Wheatere-App
![image](./cover.png)

This is A Weather App where you can search the weather by city name and see the current weather.

## Built With

- Html
- SASS
- Javascript
- webpack

## How to Run
- npm run dev
- npm install --save-dev
- http://localhost/8080
